# 장바구니 풀스택 미션 Step3

### 1. 임시 영수증 생성

- 장바구니에서 선택한 상품 id 목록으로 임시 영수증을 생성한다.
- 생성된 임시 영수증 id로 선택한 상품 목록, 주문 금액, 배송비, 총 결제 금액을 조회한다.
- 쿠폰 조회, 쿠폰 적용, 도서산간 변경, 결제 시 쿠폰 검증은 임시 영수증 기준으로 수행한다.

### 2. 쿠폰 목록

- 주문 확인 페이지의 쿠폰 적용 버튼을 클릭하면, 임시 영수증 기준으로 쿠폰 목록을 보여준다.
- 현재 상황에서 사용 불가능한 쿠폰은 선택할 수 없다.
- 최대 2개의 쿠폰까지 선택할 수 있다.
- 처음으로 쿠폰 목록을 조회하면, 조회된 쿠폰 목록 기준으로 최대 할인 쿠폰이 최초 선택된다.
- 사용하기 버튼을 클릭하면, 선택된 쿠폰을 임시 영수증에 적용하고 총 할인 금액을 보여준다.

| 쿠폰명               | 쿠폰명(영어) | 조건                           | 할인 금액                  | 만료일     |
| -------------------- | ------------ | ------------------------------ | -------------------------- | ---------- |
| 5,000원 할인 쿠폰    | FIXED5000    | 최소 주문 금액 100,000원       | 할인 5,000원               | 2026-11-30 |
| 2+1 쿠폰             | BOGO         | 동일 상품 2개 구매 시 1개 무료 | 단가 가장 높은 상품에 적용 | 2026-06-30 |
| 무료 배송 쿠폰       | FREESHIPPING | 최소 주문 금액 50,000원        | 배송비 전액                | 2026-08-31 |
| 30% 시간제 할인 쿠폰 | MIRACLESALE  | 오전 4시~7시 적용              | 주문금액의 30%             | 2026-07-31 |

### 3. 총 결제 금액 계산

- 제주 및 도서산간 지역 선택시 기본 배송비 3,000원에 추가로 3,000원이 붙어 총 배송비가 6,000원이 된다.
- 배송비 무료 쿠폰 적용 시 배송비가 0원이 된다. (도서산간 금액 포함)
- 정액 할인 쿠폰을 먼저 적용한 뒤, 정률 할인 쿠폰을 적용한다.
- 쿠폰 적용이나 도서산간 여부가 변경되면 임시 영수증의 배송비와 총 결제 금액을 다시 계산한다.

### 4. 결제하기

- 현재 서비스에서는 결제 요청 API가 없기 때문에, 결제하기 버튼 클릭 시 임시 영수증에 적용된 쿠폰의 유효성을 검증한다.
- 쿠폰 정보가 유효하지 않으면 에러를 발생시키고, 결제를 취소한다.
